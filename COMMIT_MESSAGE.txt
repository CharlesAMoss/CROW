VISUAL IMPLEMENTATION
=====================

ASCII Connector Generation (GridRow.tsx):
```typescript
const getTreeConnector = () => {
  if (level === 0) return '';
  const connector = isLastChild ? '└──' : '├──';
  const indent = '│   '.repeat(level - 1);
  return indent + connector;
};
```

Tree Mode Rendering (GridBody.tsx):
- useMemo for flattening and filtering tree data
- isLastChildMap calculation: Groups nodes by parent to determine last child status
